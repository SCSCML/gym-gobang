# gym-gobang

OpenAI Gym [gobang](https://en.wikipedia.org/wiki/Go_Bang) environment

# Dependencies

- Python3
- [gym](https://github.com/openai/gym)

# Installation

```bash
git clone https://github.com/SCSCML/gym-gobang
cd gym-gobang
pip install -e .
```

# Default configuration
- Board size : 15 by 15
- Rule : 'Free rule' without limitation.
